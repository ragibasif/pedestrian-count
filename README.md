# pedestrian-count

Pedestrian Count project for the [New York City Department of Transportation (NYC DOT)](https://www.nyc.gov/html/dot/html/home/home.shtml).

Data acquired from [NYC Open Data](https://opendata.cityofnewyork.us/).
